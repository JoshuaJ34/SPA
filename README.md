SPA for my project 1004 project

This SPA is an app that people can use to learn information about different countries. Countries can be clicked but only countries that are blue when hovered over can be clicked while red ones cannot be clicked. Not all countries have been added. A quiz can also be completed for each added country. A sgv has been used within the code to add a interactive world map. This world map was gotten from simple maps sgv which was free to use. this sgv only creates a world map with the names for each shape.
